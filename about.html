그래프 값 추출기도구로 돌아가기

서비스 소개

그래프 값 추출기는 그래프 스크린샷에서 좌표 값을 읽어 CSV 데이터로 정리하는 브라우저 기반 도구입니다.

무엇을 할 수 있나요?

이미지 위에 X축과 Y축의 기준점을 지정한 다음 점 또는 선을 클릭하면, 화면상 위치를 축 값으로 환산합니다. 숫자 축, 날짜 축, 선형 스케일, 로그 스케일을 지원하며 여러 데이터 시리즈를 구분해 관리할 수 있습니다.

데이터는 어떻게 처리되나요?

이미지 선택과 좌표 계산은 사용자의 브라우저에서 수행됩니다. 프로젝트 저장 기능은 사용자가 선택한 위치에 JSON 파일을 내려받는 방식입니다. 자세한 내용은 개인정보처리방침을 확인하세요.

피드백과 오류 제보

기능 제안, 번역 개선, 계산 오류 제보는 GitHub Issues에서 받습니다. 재현 가능한 이미지와 축 설정을 함께 알려 주시면 확인에 도움이 됩니다.
