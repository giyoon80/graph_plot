그래프 값 추출기

스크린샷을 올리고 축 기준점 4개를 찍은 다음 데이터 포인트를 클릭하세요.

사용 가이드

이미지

PNG, JPG 같은 그래프 스크린샷을 올릴 수 있습니다.

프로젝트

이미지, 축 설정, 시리즈, 데이터 포인트를 JSON 파일로 저장하고 다시 불러옵니다.

1. 축 기준점

각 축의 두 끝 기준점을 이미지에서 클릭합니다. 기준점은 축 눈금 위에 찍는 것이 좋습니다.

2. 축 값 입력

입력한 값과 이미지에서 찍은 기준점을 이용해 클릭한 점의 X/Y 값을 계산합니다.

3. 결과

이미지를 업로드하세요.

# Series X Y
4x
그래프 스크린샷을 업로드하면 여기에서 축 기준점과 데이터 포인트를 찍을 수 있습니다.

익명 피드백

이름을 공개하지 않고 의견을 남길 수 있습니다. Disqus 게스트 댓글이 켜져 있어야 로그인 없이 작성할 수 있습니다.
