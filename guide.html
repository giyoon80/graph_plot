그래프 값 추출 가이드

아래 순서대로 화면의 버튼과 입력칸을 사용하면 그래프 스크린샷을 CSV 데이터로 정리할 수 있습니다. 모든 좌표 계산은 브라우저에서 수행됩니다.

1. 이미지 선택

왼쪽 패널에서 이미지 선택을 누르고 그래프 스크린샷을 고릅니다. 축, 눈금, 값 표기가 모두 보이는 이미지를 사용하세요. 축 일부가 잘렸거나 눈금값이 보이지 않으면 정확한 값으로 환산할 수 없습니다.

이미지가 오른쪽 작업 영역에 표시되면 X축 왼쪽 모드가 자동으로 선택됩니다. 이제 축의 위치를 이미지에서 지정합니다.

2. 축 기준점 4개 지정

X축 왼쪽, X축 오른쪽, Y축 아래, Y축 위 버튼을 차례로 누르고 해당 축의 눈금선 위를 클릭합니다. 예를 들어 X축 왼쪽 값이 0인 눈금선과 X축 오른쪽 값이 10인 눈금선을 지정합니다.

3. 축 값과 스케일 입력

축 타입을 숫자 또는 날짜로 선택한 뒤 X 왼쪽 값, X 오른쪽 값, Y 아래 값, Y 위 값을 원본 그래프의 눈금값과 동일하게 입력합니다. 날짜 축은 양 끝의 실제 날짜를 입력합니다. 눈금이 일정한 간격이면 선형, 1·10·100처럼 배수로 커지면 로그를 선택하며 로그 값은 모두 0보다 커야 합니다.

4. 시리즈를 선택하고 데이터 점 찍기

결과 영역에서 시리즈 이름을 정합니다. 선이 여러 개면 추가를 눌러 시리즈를 만든 뒤 작업할 시리즈를 선택합니다. 데이터 찍기 버튼을 누르고 그래프의 점 또는 선 위를 클릭하면 표에 X/Y 값이 추가됩니다. 모든 시리즈는 동일한 X축과 Y축 기준을 공유하므로, 좌·우처럼 스케일이 다른 Y축이 있는 그래프는 축마다 별도 프로젝트로 추출하세요.

5. 점을 검증하고 결과 저장

표의 처음·중간·마지막 값을 원본 그래프와 비교합니다. 이미지 위에 마우스를 올리면 확대경이 보이며, 선택한 점은 화살표 키로 1px, Shift+화살표로 10px 움직일 수 있습니다. 잘못된 점은 마지막 삭제 또는 Delete 키로 지웁니다.

예시: 날짜별 성장 그래프 추출

월별 또는 일별 성장 그래프라면 X축 타입을 날짜로 선택합니다. 예를 들어 왼쪽 눈금이 2024-01-01이고 오른쪽 눈금이 2024-12-31이면 각각 X 왼쪽 값과 X 오른쪽 값에 입력합니다. Y축은 숫자 타입으로 두고, 아래·위 눈금값을 입력합니다.

1, 10, 100, 1,000처럼 값이 배수로 증가하는 성장 그래프는 해당 축의 스케일을 로그로 바꿉니다. 점을 세 개 정도 먼저 찍어 결과가 원본 눈금과 맞는지 확인한 다음 나머지 점을 추출하세요.

자주 묻는 질문

기준점을 바꾸면 이미 찍은 데이터 점은 어떻게 되나요?

데이터 점의 이미지 위치는 유지되고, 새 기준점과 축 값으로 X/Y 결과가 다시 계산됩니다. 기준점을 수정한 뒤 표의 값을 다시 확인하세요.

축 기준점 네 개를 모두 찍어야 하나요?

네. X축 두 개와 Y축 두 개가 모두 설정되어야 데이터 점의 X/Y 값을 계산할 수 있습니다.

프로젝트 저장 파일에는 무엇이 들어 있나요?

프로젝트 JSON에는 선택한 이미지, 축 타입·스케일·값, 기준점, 시리즈, 데이터 점이 포함됩니다. 같은 파일을 불러오면 중단한 작업을 이어갈 수 있습니다.

작업을 이어서 하려면 프로젝트 이름을 정한 뒤 프로젝트 저장으로 JSON 파일을 받으세요. 나중에 프로젝트 불러오기에서 같은 파일을 열면 이미지·축 설정·시리즈·데이터 점이 복원됩니다. 최종 결과는 CSV 다운로드 또는 복사로 내보냅니다.
