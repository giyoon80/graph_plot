그래프 값 추출 가이드

스크린샷 속 그래프를 표 형식 데이터로 바꿀 때 정확도를 높이는 방법을 정리했습니다. 이 도구는 이미지를 서버에 업로드하지 않고 브라우저에서 좌표를 계산합니다.

1. 축 기준점을 먼저 정확히 잡으세요

X축과 Y축의 두 끝값이 보이는 눈금선을 선택합니다. 축 선 자체가 기울었거나 이미지가 잘린 경우보다, 값이 분명히 표시된 눈금선 위의 교차점을 기준으로 잡는 편이 좋습니다. 두 기준점의 거리가 멀수록 클릭 오차의 영향이 줄어듭니다.

보정 후 데이터 점을 클릭하기 전에, 그래프에서 값을 이미 알고 있는 점 하나를 시험으로 찍어 결과가 맞는지 확인하세요. 어긋나면 기준점부터 다시 지정하는 것이 가장 빠릅니다.

2. 선형 스케일과 로그 스케일을 구분하세요

눈금 간격이 같은 양만큼 증가하면 선형 스케일입니다. 1, 10, 100, 1,000처럼 배수로 증가하면 로그 스케일을 선택해야 합니다. 로그 스케일에서는 기준값과 데이터값이 모두 양수여야 하며, 0 또는 음수 값이 포함된 축에는 사용할 수 없습니다.

3. 날짜 축은 표시 형식을 확인하세요

X축 또는 Y축이 날짜인 경우, 양 끝 기준값을 실제 날짜로 입력합니다. 그래프의 날짜 표시가 월/일인지 일/월인지 먼저 확인하세요. 잘못 해석하면 모든 추출 값이 일정하게 어긋납니다.

4. 여러 시리즈는 색상과 이름으로 분리하세요

한 그래프에 여러 선이나 막대가 있다면 시리즈를 추가하고 의미 있는 이름을 붙인 뒤 한 시리즈씩 추출합니다. 추출 후 표에서 시리즈별로 정렬하면 누락되거나 다른 선을 잘못 클릭한 점을 쉽게 찾을 수 있습니다.

5. CSV를 저장하기 전 검증하세요

처음·중간·마지막 지점의 값을 원본 축과 비교하세요. 확대경과 방향키를 이용해 점을 1px 단위로 조정할 수 있습니다. 검증이 끝난 데이터만 CSV로 내려받아 Excel 또는 Google Sheets에서 후속 분석에 사용하세요.

이 도구의 계산 결과는 원본 이미지의 해상도와 클릭 위치에 영향을 받습니다. 연구·재무·안전 관련 의사결정에는 원본 데이터나 공식 자료로 결과를 다시 확인하세요.
